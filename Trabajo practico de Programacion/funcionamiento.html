    FUNCIONAMIENTO

    El funcionamiento de Java se basa en un proceso que permite que los programas escritos en este lenguaje sean altamente portables y puedan ejecutarse en diferentes sistemas operativos sin modificaciones. Este proceso gira en torno a la Máquina Virtual de Java (JVM), un componente clave que actúa como intermediario entre el código Java y el hardware del sistema.

    Cuando un desarrollador escribe código fuente en Java (archivos .java), este código es compilado por el compilador de Java (javac) en un formato intermedio llamado bytecode, que se almacena en archivos .class. El bytecode no es código máquina específico de ningún procesador, sino un conjunto de instrucciones que puede ser entendido por la JVM. La JVM es responsable de cargar, verificar, interpretar y ejecutar el bytecode. Funciona como una capa de abstracción sobre el sistema operativo y el hardware, permitiendo que el mismo programa Java pueda ejecutarse en cualquier dispositivo que tenga una JVM instalada, cumpliendo así el principio de "escríbelo una vez, ejecútalo en cualquier parte" (Write Once, Run Anywhere)
