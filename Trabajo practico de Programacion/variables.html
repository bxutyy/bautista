    variables

    En Java, las variables se clasifican principalmente en tres tipos según su ámbito y uso dentro del programa: Variables locales: Se declaran dentro de un método, constructor o bloque. Solo son accesibles dentro del mismo método o bloque donde se definen. No tienen valores predeterminados, por lo que deben ser inicializadas antes de usarse. Variables de instancia: Se definen dentro de una clase pero fuera de cualquier método, sin el modificador static. Cada objeto (instancia) de la clase tiene su propia copia de estas variables. Se utilizan para representar las propiedades o atributos de un objeto. Variables de clase (estáticas): Se declaran con el modificador static dentro de una clase. Son compartidas por todas las instancias de la clase; existe una sola copia para toda la clase. Son útiles para valores o comportamientos comunes a todas las instancias. Además.
    Java tiene un tipo especial de variable llamada parámetros, que son variables que se reciben como entrada en los métodos y cuyo ámbito está limitado al método que las recibe. En cuanto a los tipos de datos que pueden almacenar las variables, Java distingue entre:
    Tipos de datos primitivos: byte (8 bits), short (16 bits), int (32 bits), long (64 bits) para números enteros. float (32 bits), double (64 bits) para números en coma flotante. char para un solo carácter Unicode (16 bits). boolean para valores de verdad (true o false). Tipos de datos no primitivos o referencias: String para cadenas de texto. Arrays, clases y objetos definidos por el usuario. Para declarar una variable en Java se usa la sintaxis: java tipoDeDato nombreVariable = valorInicial; Por ejemplo: java int edad = 30; String nombre = "Ana"; boolean esEstudiante = true;
